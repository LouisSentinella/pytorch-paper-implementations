# Deep Learning Paper Implementations

A progressive curriculum of deep learning architectures to implement in PyTorch, from foundational to cutting-edge.

---

## Foundational Architectures

### 1. LeNet-5 (1998)
Classic CNN for digit recognition
- **Paper:** [Gradient-Based Learning Applied to Document Recognition](http://yann.lecun.com/exdb/pubmed/index.html#lecun-98)

### 2. AlexNet (2012)
Modern deep learning renaissance
- **Paper:** [ImageNet Classification with Deep Convolutional Neural Networks](https://papers.nips.cc/paper/2012/hash/c399862d3b9d6b76c8436e924a68c45b-Abstract.html)

### 3. VGGNet (2014)
Depth and simplicity
- **Paper:** [Very Deep Convolutional Networks for Large-Scale Image Recognition](https://arxiv.org/abs/1409.1556)

### 4. Inception/GoogLeNet (2014)
Multi-scale feature extraction
- **Paper:** [Going Deeper with Convolutions](https://arxiv.org/abs/1409.4842)

### 5. ResNet (2015)
Residual connections (critical for modern deep learning)
- **Paper:** [Deep Residual Learning for Image Recognition](https://arxiv.org/abs/1512.03385)

### 6. DenseNet (2016)
Dense connections, memory efficient
- **Paper:** [Densely Connected Convolutional Networks](https://arxiv.org/abs/1608.06993)

---

## Segmentation & Detection

### 7. U-Net (2015)
Biomedical image segmentation
- **Paper:** [U-Net: Convolutional Networks for Biomedical Image Segmentation](https://arxiv.org/abs/1505.04597)

### 8. FCN (2015)
Fully Convolutional Networks for segmentation
- **Paper:** [Fully Convolutional Networks for Semantic Segmentation](https://arxiv.org/abs/1411.4038)

### 9. YOLO (2015)
Real-time object detection (v1 is surprisingly simple)
- **Paper:** [You Only Look Once: Unified, Real-Time Object Detection](https://arxiv.org/abs/1506.02640)

### 10. Mask R-CNN (2017)
Instance segmentation
- **Paper:** [Mask R-CNN](https://arxiv.org/abs/1703.06870)

---

## Generative Models

### 11. GAN (2014)
Foundation of generative adversarial networks
- **Paper:** [Generative Adversarial Networks](https://arxiv.org/abs/1406.2661)

### 12. DCGAN (2015)
Stable CNN-based GAN
- **Paper:** [Unsupervised Representation Learning with Deep Convolutional GANs](https://arxiv.org/abs/1511.06434)

### 13. Pix2Pix (2016)
Image-to-image translation (surprisingly elegant)
- **Paper:** [Image-to-Image Translation with Conditional Adversarial Networks](https://arxiv.org/abs/1611.07004)

### 14. CycleGAN (2017)
Unpaired image translation
- **Paper:** [Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks](https://arxiv.org/abs/1703.10593)

### 15. VAE (2013)
Variational autoencoder
- **Paper:** [Auto-Encoding Variational Bayes](https://arxiv.org/abs/1312.6114)

### 16. Diffusion Models (DDPM) (2020)
Current SOTA generation
- **Paper:** [Denoising Diffusion Probabilistic Models](https://arxiv.org/abs/2006.11239)

---

## Sequence Models

### 17. LSTM (1997)
Long short-term memory
- **Paper:** [Long Short-Term Memory](https://www.bioinf.jku.at/publications/older/2604.pdf)

### 18. GRU (2014)
Simpler alternative to LSTM
- **Paper:** [Learning Phrase Representations using RNN Encoder-Decoder](https://arxiv.org/abs/1406.1078)

### 19. Seq2Seq (2014)
Encoder-decoder architecture
- **Paper:** [Sequence to Sequence Learning with Neural Networks](https://arxiv.org/abs/1409.3215)

### 20. Attention Mechanism (2014)
Foundation for transformers
- **Paper:** [Neural Machine Translation by Jointly Learning to Align and Translate](https://arxiv.org/abs/1409.0473)

---

## Transformer Era

### 21. Transformer (2017)
Attention is all you need
- **Paper:** [Attention Is All You Need](https://arxiv.org/abs/1706.03762)

### 22. BERT (2018)
Bidirectional encoding
- **Paper:** [BERT: Pre-training of Deep Bidirectional Transformers](https://arxiv.org/abs/1810.04805)

### 23. GPT (2018)
Autoregressive language modeling
- **Paper:** [Improving Language Understanding by Generative Pre-Training](https://cdn.openai.com/research-covers/language-unsupervised/language_understanding_paper.pdf)

### 24. Vision Transformer (ViT) (2020)
Transformers for images
- **Paper:** [An Image is Worth 16x16 Words](https://arxiv.org/abs/2010.11929)

---

## Interesting/Specialized Architectures

### 25. Neural ODEs (2018)
Continuous depth networks (mathematically beautiful)
- **Paper:** [Neural Ordinary Differential Equations](https://arxiv.org/abs/1806.07366)

### 26. NeRF (2020)
Neural Radiance Fields for 3D (insanely cool)
- **Paper:** [NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis](https://arxiv.org/abs/2003.08934)

### 27. Graph Neural Networks (GCN) (2016)
For graph-structured data
- **Paper:** [Semi-Supervised Classification with Graph Convolutional Networks](https://arxiv.org/abs/1609.02907)

### 28. Neural Turing Machines (2014)
Memory-augmented networks
- **Paper:** [Neural Turing Machines](https://arxiv.org/abs/1410.5401)

### 29. Capsule Networks (2017)
Hinton's alternative to CNNs
- **Paper:** [Dynamic Routing Between Capsules](https://arxiv.org/abs/1710.09829)

### 30. CLIP (2021)
Contrastive language-image pre-training
- **Paper:** [Learning Transferable Visual Models From Natural Language Supervision](https://arxiv.org/abs/2103.00020)

### 31. SimCLR (2020)
Self-supervised contrastive learning
- **Paper:** [A Simple Framework for Contrastive Learning of Visual Representations](https://arxiv.org/abs/2002.05709)

### 32. MobileNet (2017)
Efficient mobile architectures
- **Paper:** [MobileNets: Efficient Convolutional Neural Networks for Mobile Vision Applications](https://arxiv.org/abs/1704.04861)

### 33. EfficientNet (2019)
Compound scaling
- **Paper:** [EfficientNet: Rethinking Model Scaling for Convolutional Neural Networks](https://arxiv.org/abs/1905.11946)

### 34. WaveNet (2016)
Generative audio (interesting for its dilated convolutions)
- **Paper:** [WaveNet: A Generative Model for Raw Audio](https://arxiv.org/abs/1609.03499)
